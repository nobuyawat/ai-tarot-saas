# 受講者用コピペ教材
# Udemy講座「非エンジニアがAIだけでSaaSを公開する方法」

> **この教材の使い方**
> 1. 各レクチャーの動画を見る
> 2. `📋 コピペ` のボックスをそのままコピーする
> 3. Claude Code に貼り付けてエンターを押す
> 4. AIが作業してくれるのを待つ
> 5. `✅ 確認` の手順で動作を確認する

> **🎯 マークがある箇所**では、AIから質問が来ます。
> 「回答例」を参考に、自分の言葉で答えてください。

> **コピペ⓪について（講師用）**
> コピペ⓪は「講座デモモード」のセットアップです。
> 講師が録画前に1回だけ貼り付けることで、①〜⑨の動作が安定します。
> 受講者に最初に見せるコピペ体験は**①の既存サイトリサーチ** です。

## L13.5: AIと一緒にエラーを解決する

開発中にエラーが出ることは普通です。大事なのは「エラーが出たらAIに聞く」という習慣です。

### エラーが出たときの対処法

📋 **エラー解決テンプレート（コピペ OK）**

```
このエラーが出ました。修正してください：

（ここにエラーメッセージを貼り付け）
```

> ☝️ エラーメッセージをそのままコピーして、上のテンプレートと一緒にClaude Codeに貼り付けるだけです。
> AIが原因を分析して、修正コードを自動生成してくれます。

### 画面がおかしいときの対処法

📋 **見た目の修正テンプレート（コピペ OK）**

```
以下の問題を修正してください：

（ここに問題の説明を書く）

例：
- カードをクリックしても裏返らない
- テキストがはみ出している
- ボタンが表示されていない
- スマホで表示が崩れる
```

### AIに指示するコツ

```
❌ 悪い例：「動きません」
⭕ 良い例：「カードをクリックしても裏返らない。コンソールにエラーは出ていない」

❌ 悪い例：「おかしい」
⭕ 良い例：「/login に遷移せずに、画面が真っ白になる」

❌ 悪い例：「全部やり直して」
⭕ 良い例：「ヒーローセクションの背景色だけ変えて。紫をもう少し暗くしたい」
```

> 💡 **ポイント**：エラーメッセージは「翻訳」しなくて大丈夫です。
> 英語のエラーでも、そのままAIに渡せば理解してくれます。

### よくあるエラーと対処

| エラー | 原因 | 対処法 |
|--------|------|--------|
| `command not found: claude` | Claude Code が未インストール | L3 を再確認 |
| `command not found: npm` | Node.js が未インストール | L4 を再確認 |
| `ENOENT: no such file or directory` | 間違ったフォルダにいる | `cd ai-tarot-saas` で移動 |
| `Module not found` | ファイルが足りない | エラーメッセージをAIに貼り付け |
| `localhost:3000 に接続できません` | サーバーが起動していない | `npm run dev` を実行 |
| 画面が真っ白 | ビルドエラー | ターミナルのエラーメッセージをAIに貼り付け |

## 💡 困ったときのFAQ

### Q. AIの返答が動画と少し違います
> Claude Code は毎回少しずつ異なるコードを生成します。これは正常です。
> 見た目や動作が大きく異なる場合は、追加で指示を出して調整してください。
> 「動画と同じようにして」ではなく、「こういう風にして」と具体的に伝えるのがコツです。

### Q. エラーが出て進めません
> エラーメッセージをそのまま Claude Code に貼り付けてください。
> 「このエラーが出ました。修正してください：」に続けて貼るだけで大丈夫です。
> → 詳しくは L13.5 を参照してください。

### Q. デザインが思ったものと違います
> 「背景をもっと暗くして」「ボタンをもっと大きくして」「文字色を白にして」など、
> 日本語で具体的に伝えれば、AIが調整してくれます。
> 何度でもやり直せるので、気軽に試してください。

### Q. Claude Code が途中で止まりました
> ターミナルで `Ctrl + C` を押して一度止めてから、`claude` と入力して再起動してください。
> 前回の作業内容は保存されているので、「続きをお願い」と指示すれば再開できます。

### Q. npm run dev で起動しません
> まず `cd ai-tarot-saas` で正しいフォルダにいることを確認してください。
> それでもダメな場合は、エラーメッセージをそのまま Claude Code に貼り付けてください。

### Q. Claude Code の費用が気になります
> Claude Code は月額 $20 の Pro プランで使えます。
> Vercel は無料プランで十分です。
> 合計で月約 $20（約3,000円）が唯一の費用です。
