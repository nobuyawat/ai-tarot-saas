## 講座説明（メイン）

### この講座について

「SaaSを作ってみたい。でも、プログラミングができない。」

そんな悩みを持つ方のための講座です。

この講座では、**AIだけ**を使って、本格的なSaaSアプリケーションを開発し、インターネット上に公開するまでの全プロセスを、ステップバイステップで解説します。

使うのは **Claude Code** というAI開発ツール。
日本語で「こういうアプリを作って」と指示するだけで、AIがコードを自動生成してくれます。
**コードは一行も書きません。**

### 何を作るのか？

完成物は「**AIタロット占いSaaS**」です。

- 神秘的な紫×ゴールドのデザイン
- 7枚のタロットカード（3Dフリップアニメーション付き）
- 無料占い機能
- Premiumコンテンツ（ぼかし表示のフリーミアム構造）
- Googleログイン画面（デモ）
- Stripe風決済画面（デモ）
- Vercelで世界に公開

### この講座が他と違う3つのポイント

**1. 「動くもの」ではなく「見せられるもの」を作る**

「UI Pro Max」というClaude Codeのスキルを使うことで、非エンジニアでもプロレベルのUIデザインが実現できます。グラデーション、3Dアニメーション、グロウエフェクトなど、本格的なデザインがAIによって自動生成されます。

**2. SaaSの「ビジネス構造」まで学べる**

単にアプリを作るだけでなく、フリーミアムモデル、ログイン導線、決済フローといった、SaaSビジネスに必要な構造を実際に実装します。公開時点で課金導線が組み込まれた、実用的なMVPが完成します。

**3. 実際に公開するところまでやる**

ローカルで動かして終わりではありません。GitHub連携 → Vercelデプロイまで行い、実際にURLが発行されて、世界中からアクセスできる状態にします。

### 講座の流れ「5つのドミノ」

この講座は「5つのドミノ」構成で進みます。

1. **ゴールを見る** — 完成アプリのデモを見て全体像を把握
2. **道具を揃える** — Claude CodeとNode.jsをインストール
3. **骨組みを作る** — Hello World → タロットアプリの骨格を生成
4. **魂を入れる** — UI Pro MaxでプロUIを実現 → 占い・Premium・ログイン・決済を追加
5. **世界に届ける** — GitHub + Vercelで全世界に公開

## 学習内容（箇条書き）

この講座を修了すると、以下ができるようになります：

- Claude Codeを使って、日本語の指示だけでウェブアプリを開発できる
- UI Pro Maxスキルを活用して、プロレベルのUIデザインをAIに自動生成させられる
- Next.jsベースのSaaSアプリケーションの基本構造を理解できる
- フリーミアムモデル（無料機能 + Premium有料機能）の設計ができる
- ログイン → 決済 → 完了という課金フローの実装方法を理解できる
- GitHubとVercelを使って、アプリをインターネットに公開できる
- MVPの考え方を理解し、最小限の機能で素早くリリースする方法を学べる

## 対象者

### この講座が向いている人

- プログラミング未経験だけど、自分のSaaSを作ってみたい人
- AIツールに興味があり、実際にプロダクトを作りたい人
- 副業・個人開発でSaaSビジネスを始めたい人
- Claude Codeの実践的な使い方を学びたい人
- 「アイデアはあるけど、技術的なハードルで諦めていた」人
- 非エンジニアのディレクター・マーケターでプロトタイプを作りたい人

### この講座が向いていない人

- すでにプログラミングができるエンジニアの方（物足りない可能性があります）
- 本格的なバックエンド開発やデータベース設計を学びたい方
- 実際のStripe決済やGoogle OAuth認証の実装を学びたい方（本講座はデモ実装です）
