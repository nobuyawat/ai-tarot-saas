## L3: Claude Code導入

**[スライド 4]**

皆さん、ドミノ2「道具を揃える」の最初のステップです。
ここでは**Claude Code**をインストールします。
Claude Codeは、皆さんのAI開発パートナーになるツールです。

まず、Anthropicのウェブサイトにアクセスして、
アカウントを作成します。

**[画面操作]** ※anthropic.comを表示

anthropic.comにアクセスして、
サインアップボタンをクリックしてください。
メールアドレスとパスワードを入力して、
アカウントを作成します。

次に、**Proプラン**に加入します。
Claude Codeを使うには、
月額20ドルのProプランが必要です。
これが唯一の費用です。
後で使うVercelは無料プランで十分なので、
実質かかるのはこの20ドルだけです。

さて、インストールに移りましょう。
ターミナルを開きます。
Macの方はSpotlight検索で「ターミナル」と入力してください。
Windowsの方はPowerShellを開いてください。

ターミナルが開いたら、
インストールコマンドを1つ入力するだけです。
画面に表示されているコマンドをそのまま入力して
エンターキーを押すと、
自動的にインストールが始まります。
数分で完了します。

インストールが終わったら、
ターミナルに「claude --version」と入力してください。
バージョン番号が表示されたら成功です。

最後に「claude」と入力して起動し、
「こんにちは」と話しかけてみてください。
AIから返事が返ってくれば、セットアップ完了です！

[間]

これが今後ずっと皆さんのパートナーになるツールです。
次のレクチャーでは、
もう1つの道具、Node.jsをインストールします。

## L4: Node.js導入

**[スライド 5]**

皆さん、道具を揃える2つ目のステップです。
今回は**Node.js**をインストールします。

Node.jsは、アプリを動かすためのエンジンのようなものです。
車にエンジンが必要なように、
ウェブアプリにはNode.jsが必要です。

**[画面操作]** ※nodejs.orgを表示

まず、nodejs.orgにアクセスしてください。
トップページに2つのダウンロードボタンが表示されます。
左側の「**LTS**」と書かれたボタンをクリックしてください。
LTSは安定版という意味です。

ダウンロードされたインストーラーを開いて、
基本的に「次へ」ボタンを連打するだけでOKです。
設定を変える必要はありません。

インストールが完了したら、
ターミナルを開いて「node --version」と入力してください。
バージョン番号が表示されたら成功です。

続けて「npm --version」も確認してください。
npmはNode.jsと一緒にインストールされる
パッケージ管理ツールです。

なぜNode.jsが必要なのか、簡単に説明します。
この講座で作るアプリは
Next.jsというフレームワークを使います。
Next.jsはNode.jsの上で動くので、
Node.jsが必要なんです。

でも、詳しい仕組みを理解する必要はありません。
「アプリを動かすのに必要な土台」と
思っていただければ十分です。

もしエラーが出た場合は、
まずパソコンを再起動してみてください。

[間]

これで道具が2つ揃いました！
Claude CodeとNode.js。
次のレクチャーからは、
いよいよドミノ3「骨組みを作る」に入ります。
楽しみにしていてください！

スキルとは何か？
AIに**特定の得意分野**を持たせる機能のことです。

普通のAIは家庭料理レベル。

## L11: Premium導線

**[スライド 13]**

皆さん、ここからが**SaaSらしくなる瞬間**です。

先ほど作った無料鑑定の下に、
もう1つの層を追加します。
それが「**Premium鑑定**」です。

フリーミアムモデルという言葉を
聞いたことはありますか？
無料で基本機能を使えるようにして、
より深い価値は有料で提供する。
多くの成功しているSaaSがこのモデルを採用しています。

**[画面操作]** ※Claude Code操作

具体的にどう実装するか。
Claude Codeにこう指示します。

「**各カードのデータに、
paidReadingフィールドを追加して。
無料鑑定の下にPremiumセクションを表示し、
詳細鑑定のテキストをブラーで隠して、
¥300で解放するボタンを配置して**」

[間]

AIがブラー、つまりぼかしエフェクトのかかった
Premiumセクションを生成してくれます。

ユーザーからは、
そこに何か魅力的な情報があることは分かるけれど、
内容は読めない。
「もっと知りたい」という気持ちを
自然に引き出す仕掛けです。

そしてここがポイント。
このPremiumボタン、
今はまだ決済システムにはつながっていません。

実際のSaaSと同じユーザー体験の流れを、
