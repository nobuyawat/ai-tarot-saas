## コピペ指示文の配置方法

Udemyでは以下の2つの方法で指示文を配置できます：

### 方法A: レクチャー説明文に直接記載（推奨）

```
各レクチャーの「説明」欄に、コピペ指示文をそのまま記載する。
受講者は動画を見ながら、すぐ下の説明欄からコピーできる。
```

**メリット**: 一画面で完結。受講者がファイルを開く手間がない。
**注意**: Udemyの説明欄はMarkdown非対応。プレーンテキストで記載する。

### 方法B: リソースファイルとしてダウンロード配布

```
コピペ指示文をテキストファイル（.txt）にまとめて、
各レクチャーの「リソース」からダウンロード可能にする。
```

**メリット**: ファイルとして手元に残る。オフラインでも参照可能。
**注意**: ダウンロードの手間がかかる。

### 推奨: 方法A + B の併用

- **説明欄**: コピペ指示文をそのまま記載（すぐ使える）
- **リソース**: 全レクチャーの指示文をまとめた PDF を1ファイル配布（まとめ参照用）

## スライド・動画 制作チェックリスト

### 各レクチャー共通

```
□ スライド: テーマに合ったビジュアル（紫×ゴールド推奨）
□ 動画: 画面収録 + 顔出し or 音声ナレーション
□ 説明文: コピペ指示文を配置（該当レクチャーのみ）
□ リソース: 補足資料があれば添付
```

### コピペ指示文があるレクチャー

```
□ 説明文にコピペ指示文を記載
□ 動画内でも「説明欄からコピーしてください」と案内
□ 質問ポイント（L6, L9）は回答例も一緒に記載
□ 動作確認チェックリストを添付
```

### 動画撮影の優先順

```
1. L1  （完成版デモ → 最初の印象が大事）
2. L5  （Hello World → 最初の感動体験）
3. L8  （Before/After → 視覚的インパクト大）
4. L12 （全フロー完成 → 達成感のピーク）
5. L14 （Vercel公開 → フィナーレ）
6. 残りのレクチャー
```
